# Lemuel Layola Apa Portfolio

Static portfolio site prepared for GitHub Pages.

## Publish To GitHub Pages

1. Create a new GitHub repository.
2. Push the contents of this folder to the repository.
3. In GitHub, open `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Set the source to the `main` branch and the `/ (root)` folder.
6. Save and wait for GitHub Pages to publish the site.

Your published URL will usually be:

- `https://<your-github-username>.github.io/<repository-name>/`

If you name the repository `<your-github-username>.github.io`, the site URL will be:

- `https://<your-github-username>.github.io/`

## Local Structure

- `index.html` - main portfolio page
- `images/` - local image assets used by the portfolio
- `.nojekyll` - ensures GitHub Pages serves files as a plain static site
